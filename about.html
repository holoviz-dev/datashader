About Us¶

Datashader is part of the HoloViz ecosystem for making browser-based data visualization in Python easier to use, easier to learn, and more powerful. See holoviz.org for related packages that you can use with Datashader and status.holoviz.org for the current status of each HoloViz project.

Datashader is supported and maintained by Anaconda.
